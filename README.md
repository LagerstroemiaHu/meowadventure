
# 圆头耄耋：喵生冒险 (Round-headed Cat's Journey)

这是一个基于 React + TailwindCSS 开发的文字冒险游戏（Gamebook），模拟一只猫在 15 天内的生存与晋升之路。

---

## 🏆 结局与成就图鉴 (Endings & Legends)

游戏共有 **30+** 种结局和成就，取决于你的生存天数、属性状态以及剧情选择。

### 🌟 传奇结局 (Victory Endings)
成功存活至第 15 天结束，根据最终所处的阶段判定。

| 结局代码 | 结局名称 | 达成条件 |
| :--- | :--- | :--- |
| `SURVIVOR_STRAY` | 街头幸存者 | 第15天仍处于 **流浪** 阶段。 |
| `LEGEND_LORD` | 不朽教父 | 第15天处于 **猫王** 阶段。 |
| `LEGEND_COMFORT` | 养老赢家 | 第15天处于 **豪宅** 阶段。 |
| `LEGEND_SUPERSTAR` | 喵星巨星 | 第15天处于 **网红** 阶段 (完美通关)。 |

### 🎭 属性评价 (Traits)
基于最终属性数值的评价。

*   **爱因斯坦** (`STAT_MAX_SMARTS`): 智力 >= 95
*   **哥斯拉** (`STAT_MAX_HISSING`): 哈气 >= 95
*   **六边形战士** (`STAT_BALANCED`): 四维属性均 >= 60
